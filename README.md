# Node Starter
A basic nodejs project starter

## Build
`yarn install` or `npm install`

## Run
`yarn start` or `npm start`

## Test
To run unit tests:
`yarn test` or `npm test`

For continuous test running:
`npm run test:watch`

For running tests with coverage:
`npm run test:coverage`

Lint code:
`npm run lint`

# Docker
`make docker-build`
